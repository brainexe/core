# Features

##ServiceAnnotations
   - @Command
   - @Controller (using @Route and @Guest method annotation)
   - @EventListener
   - @Middleware
   - Twig Extension
 - Redis database
 - DIC
 - Background Events via Message Queue
 - SMS Gateway
 - Request middlewares
   - Authentication (set current user)
   - Locale
   - Cache
   - Generation time log
   - ...
 - Expression language
 - Input control (define commands via regexp. e.g. "send mail to .*? with subject '.*?' and body '.*?'")
 - Arduino API
